Instruction to play game

  • From motion tab drag and drop the left.right,up,down blocks to workspace and connect them
  • From Loop tab drag the loop and select the motion that you reaptedly want to perfom and
  • from math tab drag the number and write the number in the block to specify how many times you want to execute loop
  • Select the background from drop down menu and click on change to change the background of the table
  • Click on run button to run the object on screen
